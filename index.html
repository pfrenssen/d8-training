First things first

mymodule.info -> mymodule.info.yml

Drupal 7:


name = Example
description = My module.
core = 7.x
files[] = myfile.php
dependencies[] = views

Drupal 8:


name: Example
type: module
description: 'My module.'
core: 8.x
dependencies:
  - views
